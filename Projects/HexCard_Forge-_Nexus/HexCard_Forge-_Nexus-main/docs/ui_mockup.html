Project Overview

HexCard Forge is a hybrid desktop and web application where players combine hex-shaped cards to forge new characters, build or generate hex maps, and battle in a strategic turn-based game. The entire experience shares one data model and runs from a single code base using React, Three.js and Tauri.

  • Single-page navigation between Forge, Map and Play modes.
  • Works offline as a compact Tauri desktop app and online in the browser.
  • Data validated with Zod and synchronized through a global Zustand store.
  • Procedural map generation and AI-assisted card fusion planned.

Forge Screen

3D crucible – drag cards here

The Forge lets you drag six hex-cards onto sockets around a glowing crucible. As cards are placed, live stats and preview abilities appear. Clicking Forge creates a new character stored in your collection.

Map Editor

Hex grid with terrain tools

Design battlefields or craft them procedurally. Use brushes to paint terrain, place props and spawn points, or enter a seed to generate a map automatically. Maps can be exported as JSON or saved directly into your project.

Play Mode

Turn-based hex battle board

Deploy forged characters on the hex board in turn-based combat. All moves and actions flow through the shared engine for consistent game rules and undo/redo support.

Technology Stack

  • React 18 with Vite for a fast development experience.
  • Three.js and Pixi.js for 3D scenes and HUD rendering.
  • Zustand state management with persistence.
  • Zod schemas for strict runtime validation.
  • Tauri for lightweight cross-platform desktop builds.
